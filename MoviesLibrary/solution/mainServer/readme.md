# Film Chat Web App

This is a full-stack web application built with **Node.js**, **Express**, **Socket.IO**, and **Handlebars**.  
It allows users to search movies, view details, read reviews, and participate in real-time chat rooms related to each movie.

---

## Features

- Search for movies by title and category
- View detailed information and user reviews
- Real-time chat room for each movie (Socket.IO)
- Scrollable review carousel with "Load more"
- Pagination for search results
- Swagger UI API documentation at `/api-docs`

---

## Prerequisites

- **Node.js** 
- **npm** (Node Package Manager)
- **MongoDB** and **PostgreSQL**
