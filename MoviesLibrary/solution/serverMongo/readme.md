# Movies API

This is a RESTful API built with Node.js, Express, and MongoDB (using Mongoose).  
It provides endpoints to fetch data related to movies such as reviews, releases, posters, languages, studios, and themes.

---

## Features

- Get all reviews for a given movie
- Get all release info for a given movie
- Get poster info for a given movie
- Get available languages for a movie
- Get studios involved in a movie
- Get themes of a movie
- Swagger UI documentation available at `/api-docs`

---

## Prerequisites

- Node.js 
- MongoDB (running locally or remotely)
- npm (Node Package Manager)
